# 최적화
- 웹 서비스의 성능을 개선하는 모든 행위를 일컫음
- 아주 단순한 것부터 아주 어려운 방법까지 매우 다양함
  - ex)
  - 서버의 응답속도 개선
  - 이미지, 폰트, 코드 파일 등의 정적 파일 로딩 개선

### useMemo
- `메모리제이션` 기법을 기반으로 불 필요한 연산을 최적화 하는 리액트 훅
- 반복적으로 수행되는 동일한 연산을 기억

### React.memo
- 컴포넌트를 인수로 받아, 최적화된 컴포넌트로 만들어 반환
```jsx
const MemoizedComponent = memo(Component)
```

### useCallback
- useCallback을 사용하면 함수가 처음 생성될 때 한 번만 생성되며, 나중에는 동일한 함수 인스턴스를 재사용하게 된다.
